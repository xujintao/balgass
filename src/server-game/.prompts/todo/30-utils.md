# 30. 基础工具系统

本模块覆盖语言包、随机数、数学工具、路径/配置辅助、通用解析辅助和基础诊断 helper。基础工具系统不拥有业务规则，不直接修改玩家、对象、道具、地图、经济或运营状态；它只提供可复用的底层能力，供运行时、对象、地图、掉落、技能、脚本、运营等系统调用。

明确排除：`MuHelper`、`MuBot`、`OfflineLevelling` 归候选“助手挂机系统”；`BotSystem`、测试 Bot、模拟连接归候选“Bot/模拟玩家系统”；运营后台、GM 命令、封禁审计归 `29-ops.md`；Lua/WZ/SMD 的业务脚本生命周期归 `26-script.md`。
