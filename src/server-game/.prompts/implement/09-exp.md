你现在要为 Go 项目 `server-game` 实现并补强“经验系统”。

## 背景

你现在位于 Go 项目 `server-game` 的仓库根目录：

/home/pi/balgass/src/server-game

另有 C++ 参考项目 `GameServer`：

/home/pi/balgass-igc/igc/9.5.1.15/source/GameServer

本模块覆盖经验表、经验倍率、怪物击杀经验、普通升级、大师经验、大师升级、组队经验、经验附加来源、经验扣减、协议下发和测试。具体经验公式归公式系统，本模块只记录经验流程、状态变化、倍率应用和协议边界。

本提示词来自 `.prompts/todo/09-exp.md`，但不是要求一次实现全部 TODO。你需要把 TODO 转换成一次可闭环、可测试、风险可控的第一阶段开发任务。

## 模块边界

本模块只处理“经验系统”第一阶段所需的核心能力。跨系统能力只能调用对应系统或预留接口，不要顺手实现其他大模块。

本模块涉及的主要子模块分类包括：经验表与基础配置、ExpManager 倍率配置、击杀经验入口、击杀经验倍率叠加、普通经验增加与升级、普通升级奖励、大师经验与大师升级、组队经验。

## 第一阶段目标

完成经验来源、杀怪经验、等级提升和经验结果下发的最小闭环。

第一阶段必须能独立验收。不要试图一次性实现 `.prompts/todo/09-exp.md` 中所有条目。

- 不修改 `handle` 层协议表，除非第一阶段目标明确需要接入已存在协议且没有其他入口。
